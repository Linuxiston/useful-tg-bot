How to run?
==
- clone this repository
- create virtualenv `python3 -m venv .venv`
- run `source .venv/bin/activate`
- run `pip install -r requirements.txt`
- Rename `.env.example` to `.env`
- Paste your Bot token to `.env` file
- run `python3 main.py`
